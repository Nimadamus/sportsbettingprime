        HOW TO UPLOAD TO REALAIGIRLS.COM - SUPER SIMPLE!
═══════════════════════════════════════════════════════════════════

After running the script, you'll have a folder on your Desktop:
    "REALIAIGRILS NEW PAGES"

This folder contains 3 folders:
    ✓ images/       (your optimized images)
    ✓ thumbnails/   (smaller preview images)
    ✓ pages/        (gallery HTML files)

───────────────────────────────────────────────────────────────────
METHOD 1: FILEZILLA (EASIEST - RECOMMENDED)
───────────────────────────────────────────────────────────────────

STEP 1: Download FileZilla (Free FTP Client)
    → https://filezilla-project.org/download.php?type=client
    → Install it

STEP 2: Connect to Your Website
    Click: File → Site Manager → New Site

    Enter these details:
        Protocol:   FTP
        Host:       ftp.realaigirls.com
        Port:       21
        User:       [Your GoDaddy FTP username]
        Password:   [Your GoDaddy FTP password]

    Click: Connect

STEP 3: Upload the Folders
    LEFT SIDE (Your Computer):
        → Navigate to: Desktop/REALIAIGRILS NEW PAGES/

    RIGHT SIDE (Your Website):
        → Navigate to: /public_html/

    DRAG these 3 folders from LEFT to RIGHT:
        ✓ images/
        ✓ thumbnails/
        ✓ pages/

STEP 4: Done!
    Visit: https://realaigirls.com/pages/

───────────────────────────────────────────────────────────────────
METHOD 2: GODADDY CPANEL FILE MANAGER
───────────────────────────────────────────────────────────────────

STEP 1: Login to GoDaddy
    → https://www.godaddy.com/
    → Login with your account

STEP 2: Open cPanel
    → My Products
    → Web Hosting
    → Click "Manage" next to your hosting
    → Click "cPanel Admin"

STEP 3: Open File Manager
    → Find "Files" section
    → Click "File Manager"
    → Navigate to: public_html

STEP 4: Upload Folders
    → Click "Upload" button (top menu)
    → Select all files from Desktop/REALIAIGRILS NEW PAGES/
    → Wait for upload to complete

STEP 5: Done!
    Visit: https://realaigirls.com/pages/

───────────────────────────────────────────────────────────────────
NEED YOUR FTP CREDENTIALS?
───────────────────────────────────────────────────────────────────

If you don't know your FTP username/password:

1. Login to GoDaddy.com
2. Go to: My Products → Web Hosting
3. Click "Manage"
4. Find "FTP" section or "Hosting Settings"
5. Your FTP username will be shown
6. Click "Reset Password" if needed

───────────────────────────────────────────────────────────────────
VERIFY IT WORKED
───────────────────────────────────────────────────────────────────

After uploading, visit:
    https://realaigirls.com/pages/

You should see your gallery!

If you see "404 Not Found":
    ✗ Files may not be in the right location
    ✓ Make sure folders are in /public_html/
    ✓ Check that the paths are correct

───────────────────────────────────────────────────────────────────
TROUBLESHOOTING
───────────────────────────────────────────────────────────────────

Problem: "Can't connect to FTP"
Solution:
    • Check your FTP username/password
    • Make sure you're using ftp.realaigirls.com as host
    • Try port 21 (standard FTP)
    • Contact GoDaddy support if needed

Problem: "Images not showing"
Solution:
    • Make sure all 3 folders uploaded completely
    • Check folder permissions (should be 755)
    • Verify paths: /public_html/images/, /public_html/thumbnails/, /public_html/pages/

Problem: "Upload is slow"
Solution:
    • FTP uploads can take 10-30 minutes for lots of images
    • Be patient and let it finish
    • Don't close FileZilla until complete
