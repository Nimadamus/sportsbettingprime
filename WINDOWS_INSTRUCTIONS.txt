================================================================================
                    HOW TO RUN IN WINDOWS POWERSHELL
================================================================================

STEP 1: Open PowerShell
================================================================================

Option A - From Desktop:
  1. Hold Shift key
  2. Right-click on Desktop folder
  3. Click "Open PowerShell window here"

Option B - From Start Menu:
  1. Click Start
  2. Type "PowerShell"
  3. Right-click "Windows PowerShell"
  4. Click "Run as Administrator"
  5. Type: cd Desktop
  6. Press Enter

--------------------------------------------------------------------------------

STEP 2: Run the Scraper
================================================================================

In PowerShell, type this EXACTLY:

  .\run-covers.ps1

Then press Enter.

If you get an error about "execution policy", run this first:

  Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Bypass

Then try again:

  .\run-covers.ps1

--------------------------------------------------------------------------------

STEP 3: Alternative (If PowerShell script doesn't work)
================================================================================

Just run Python directly:

  python quick_covers_scraper.py

That's it! No .\ needed, just the command above.
